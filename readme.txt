This project provides empirical and statistical analysis on the Oracle of Bacon
actor network. The runnable files are a visualizer in src/GraphVisualizer.java
and our tests in test/NetworkTest.java. Full writeups can be found in the
writeups folder.

technical_writeup explains all functions - what the visualizer can do, as well
as all of the backend framework and analysis abilities

analysis_questions provides empirical analysis based on our tests, graphs, and
statistics.
